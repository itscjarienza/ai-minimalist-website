Icons & Illustration System

The AI Minimalist — Visual Language

00 — Style Specification
Stroke Weight
1px for icons
0.9–1.25px for illustrations
Never exceed 1.5px
Line Cap
Round caps on open paths
Square on structural lines
Consistent per composition
Fill
Nodes only — small filled circles
No filled shapes in icons
Opacity for depth: 30–60%
Grid
24×24px icon grid
Illustrations: flexible
Always square viewBox
Radius
Zero. No rounded corners.
Circles for nodes only.
Everything else is line + angle.
Color
#242424 on light
#ede2c7 on dark
Opacity for secondary elements
